Happy Birthday site for Si Thu Aung

This site includes an in-browser piano background implemented with the WebAudio API. The piano starts after the first user interaction (browsers often block autoplay). Use the `Mute` button to toggle sound.

If you'd prefer to use an MP3 file instead, place `music.mp3` at `assets/music/music.mp3` and replace or remove the WebAudio code in `script.js`.

To run:
1. Add images to `assets/photos/` named `photo1.jpg`, `photo2.jpg`, `photo3.jpg`.
2. Open `index.html` in a browser.
